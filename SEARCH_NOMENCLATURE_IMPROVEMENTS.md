# 🔍 Улучшения поиска по номенклатуре

## 📋 Обзор изменений

Реализованы значительные улучшения поиска по номенклатуре для повышения релевантности результатов при поиске точных значений, таких как марки бетона (например, "В25").

## 🎯 Основные улучшения

### 1. Увеличен boost для точных совпадений

- **`name.exact`**: boost увеличен с 6 до 15 для средних запросов
- **`match_phrase`**: boost увеличен с 12 до 20 для длинных запросов
- **Специальная логика для марок бетона**: boost до 50 для точных совпадений

### 2. Улучшенные синонимы

```typescript
// Старые синонимы
'бетон,цемент,раствор,смесь'

// Новые синонимы
'бетон,цемент,смесь' // Убрали раствор
'раствор,строительный раствор,цементный раствор' // Отдельная группа
```

### 3. Специальные синонимы для марок бетона

Добавлены специальные синонимы для популярных марок:

```typescript
'в25,в-25,бетон в25,бетон в-25,марка в25',
'в30,в-30,бетон в30,бетон в-30,марка в30',
'в35,в-35,бетон в35,бетон в-35,марка в35',
'в40,в-40,бетон в40,бетон в-40,марка в40',
'м300,м-300,марка 300,бетон м300',
'м400,м-400,марка 400,бетон м400',
'м500,м-500,марка 500,бетон м500',
```

### 4. Улучшенный function_score

Добавлен специальный бустинг для документов с марками бетона:

```typescript
{
  filter: {
    bool: {
      should: [
        { regexp: { 'name.exact': '.*в[0-9]+.*' } },
        { regexp: { 'name.exact': '.*м[0-9]+.*' } },
        { regexp: { 'name.exact': '.*марка.*[0-9]+.*' } },
      ],
    },
  },
  weight: 2.0, // Высокий бустинг для марок бетона
}
```

### 5. Специальная логика для марок бетона

Реализован специальный метод `buildConcreteGradeSearchQuery()` для обработки запросов с марками бетона:

- **Точное совпадение**: boost 50
- **Фраза**: boost 30
- **Различные форматы**: boost 15-25
- **Wildcard**: boost 10

## 🚀 Как применить улучшения

### 1. Переиндексация данных

```bash
# Запуск скрипта переиндексации
npm run ts-node src/scripts/reindexWithImprovedScoring.ts
```

### 2. Тестирование улучшений

```bash
# Запуск тестов
npm run ts-node src/scripts/testSearchImprovements.ts
```

### 3. Проверка результатов

После переиндексации поиск по маркам бетона должен показывать:

- ✅ Точные совпадения на первых позициях
- ✅ Высокие scores для релевантных результатов
- ❌ Минимум нерелевантных позиций (растворы)

## 🔧 Технические детали

### Регулярные выражения для марок бетона

```typescript
// Проверка на марку бетона
const isConcreteGrade =
	/^[вм][0-9]+$/i.test(originalQuery) ||
	/^марка\s*[вм][0-9]+$/i.test(originalQuery)
```

### Поддерживаемые форматы

- `В25`, `В30`, `В35`, `В40`
- `М300`, `М400`, `М500`
- `марка В25`, `марка М300`

Установлен `min_score: 0.5` для фильтрации нерелевантных результатов.
