# Кнопка копирования для полей формы

## Описание

Добавлена функциональность копирования значений полей формы в буфер обмена. Кнопка копирования отображается рядом с каждым полем ввода и позволяет быстро скопировать текущее значение поля.

## Компоненты

### CopyButton

Основной компонент кнопки копирования:

```tsx
<CopyButton
	value={value} // Значение для копирования
	disabled={false} // Отключить кнопку
	size='small' // Размер кнопки
	compact={false} // Компактный режим
	isMobile={false} // Мобильный режим
/>
```

### Интеграция в поля ввода

Кнопка копирования интегрирована во все типы полей ввода:

- **TextInput** - текстовые поля
- **NumberInput** - числовые поля
- **SelectInput** - выпадающие списки
- **TextareaInput** - многострочные текстовые поля
- **AutocompleteInput** - поля с автодополнением

## Использование

### Включение кнопки копирования

Кнопка копирования включается через параметр `showCopyButton={true}` в компоненте `FormField`:

```tsx
<FormField
	field={field}
	value={value}
	onChange={onChange}
	showCopyButton={true} // Включить кнопку копирования
/>
```

### В формах

В компонентах форм кнопка копирования включается автоматически:

- **BetoneForm** - основная форма заказа бетона
- **SimpleMobileBetoneForm** - мобильная версия формы
- **MobileBetoneForm** - расширенная мобильная версия

## Функциональность

### Копирование значений

- Копирует текущее значение поля в буфер обмена
- Поддерживает все типы данных (строки, числа, объекты)
- Преобразует значения в строковый формат для копирования

### Уведомления

- Показывает уведомление об успешном копировании
- Отображает ошибки при неудачном копировании
- Автоматически скрывает уведомления через 2-3 секунды

### Адаптивность

- Адаптируется под размер экрана (мобильный/десктоп)
- Поддерживает компактный режим отображения
- Оптимизирована для сенсорных экранов

## Технические детали

### Clipboard API

Использует современный Clipboard API с fallback для старых браузеров:

```javascript
// Современный API
await navigator.clipboard.writeText(textToCopy)

// Fallback для старых браузеров
document.execCommand('copy')
```

### Безопасность

- Работает только в безопасных контекстах (HTTPS)
- Проверяет доступность Clipboard API
- Обрабатывает ошибки копирования

### Производительность

- Мемоизированные компоненты для оптимизации рендеринга
- Минимальные ре-рендеры при изменении значений
- Оптимизированные обработчики событий

## Стилизация

### Размеры

- **Мобильный**: 16px иконка, 4px padding
- **Компактный**: 18px иконка, 6px padding
- **Обычный**: 20px иконка, 8px padding

### Позиционирование

- Располагается справа от поля ввода
- Выравнивается по верхнему краю поля
- Отступ 4px от поля ввода

### Состояния

- **Обычное**: стандартная иконка копирования
- **Hover**: изменение фона при наведении
- **Disabled**: полупрозрачная иконка для пустых полей

## Примеры использования

### Базовое использование

```tsx
import { CopyButton } from './components/CopyButton'

function MyForm() {
	const [value, setValue] = useState('')

	return (
		<div>
			<input value={value} onChange={e => setValue(e.target.value)} />
			<CopyButton value={value} />
		</div>
	)
}
```

### В составе поля формы

```tsx
<FormField
	field={{
		name: 'company',
		label: 'Компания',
		type: 'text',
		required: true,
	}}
	value={formData.company}
	onChange={(name, value) => setFormData(prev => ({ ...prev, [name]: value }))}
	showCopyButton={true}
/>
```

## Совместимость

- **Браузеры**: Chrome 66+, Firefox 63+, Safari 13.1+, Edge 79+
- **Мобильные**: iOS Safari 13.1+, Chrome Mobile 66+
- **React**: 16.8+ (требует хуки)
- **Material-UI**: 5.0+

## Ограничения

- Не работает в небезопасных контекстах (HTTP)
- Требует пользовательского взаимодействия для активации
- Некоторые браузеры могут блокировать автоматическое копирование
