# FormEditor

Компонент для создания и редактирования форм с интеграцией Битрикс24.

## Структура

```
FormEditor/
├── index.tsx              # Главный компонент
├── types.ts               # TypeScript типы
├── constants.ts           # Константы
├── README.md             # Документация
├── components/           # UI компоненты
│   ├── FormHeader.tsx    # Заголовок с индикаторами
│   ├── FormSettings.tsx  # Настройки формы
│   └── FieldsList.tsx    # Список полей
├── hooks/                # Бизнес-логика
│   ├── useFormEditor.ts  # Основная логика
│   ├── useDragAndDrop.ts # Перетаскивание
│   └── useFieldManagement.ts # Управление полями
└── utils/                # Утилиты (пока пусто)
```

## Принципы архитектуры

### 🎯 Единственная ответственность
- **index.tsx** - только композиция компонентов
- **hooks/** - вся бизнес-логика
- **components/** - только UI без логики
- **types.ts** - все интерфейсы
- **constants.ts** - статические данные

### 🔄 Переиспользование
- Хуки можно использовать в других компонентах
- UI компоненты независимы от логики
- Типы переиспользуются между файлами

### 📦 Размер файлов
- ✅ Все файлы < 200 строк
- ✅ Логика разделена по хукам
- ✅ UI разделен по компонентам

## Использование

```tsx
import FormEditor from './components/admin/FormEditor';

<FormEditor 
  form={existingForm} 
  onSave={handleSave} 
/>
```

## Особенности

- **Автосохранение** через 3 секунды
- **Drag & Drop** для изменения порядка полей
- **Валидация** обязательных полей
- **Интеграция с Битрикс24**
- **Responsive дизайн**
- **Индикаторы состояния**
