Размеры и файлы

  • logo.svg - основной логотип (200x200) - шестиугольник
  • favicon.svg - фавикон (32x32)
  • favicon-16.svg - для создания ICO (16x16)
  • logo192.svg - для PWA (192x192)
  • logo512.svg - для PWA (512x512)
  • logo-alt1.svg...logo-alt4.svg - альтернативные варианты

Особенности дизайна

  • Шестиугольная форма - символизирует прочность конструкций
  • Двойной контур - внешний и внутренний шестиугольники
  • Центрированная буква "Б" - четкая типографика
  • Адаптивность - логотип корректно масштабируется

Мобильная адаптация

На мобильных устройствах:

  • Логотип уменьшается до 32px
  • Текст "БетонЭкспресс" скрывается
  • Показывается сокращение "БЭ"
  • Навигация сворачивается в боковое меню
