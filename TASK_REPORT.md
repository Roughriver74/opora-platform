## 🎯 Обзор проекта

Разработана полнофункциональная CRM система для оформления заявок на покупку бетона с интеграцией Битрикс24. Система включает публичную форму заказа, административную панель, систему управления пользователями и заявками.

- **190+ полей формы** с полной интеграцией Битрикс24

- **Мониторинг производительности** в реальном времени

## 🛠️ Технический стек

### Frontend

- **React 18** с TypeScript
- **Material-UI** для компонентов
- **React Hook Form** для управления формами
- **React Router** для навигации
- **Axios** для API запросов

### Backend

- **Node.js** с Express.js
- **TypeScript** для типизации
- **MongoDB** с Mongoose ODM
- **JWT** для авторизации
- **PM2** для управления процессами

### Интеграции

- **Битрикс24 REST API** для CRM
- **Nginx** для проксирования
- **Docker** для контейнеризации
