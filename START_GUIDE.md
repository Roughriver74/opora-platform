# 🚀 Руководство по запуску Beton CRM

## Быстрый запуск

### Вариант 1: Автоматический запуск (рекомендуется)

```bash
npm start
```

### Вариант 2: Раздельный запуск

Откройте два терминала:

**Терминал 1 - Сервер:**

```bash
npm run dev:server
```

**Терминал 2 - Клиент:**

```bash
npm run dev:client
```

## Проверка работы

- 🌐 **Клиент**: http://localhost:3000
- 📡 **Сервер API**: http://localhost:5001

## Остановка приложения

Нажмите `Ctrl+C` в терминале для остановки процессов.

## Устранение проблем

### Порт уже занят

```bash
# Найти процессы на портах
lsof -ti:3000
lsof -ti:5001

# Остановить процессы
kill -9 <PID>
```

### Проблемы с зависимостями

```bash
# Переустановить все зависимости
npm run install:all
```

### Очистка кэша

```bash
# Очистить кэш npm
npm cache clean --force

# Удалить node_modules и переустановить
rm -rf node_modules client/node_modules server/node_modules
npm run install:all
```
