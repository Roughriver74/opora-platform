# Миграция Beton CRM с MongoDB на PostgreSQL

## Выполненные работы

### 1. Создание новой архитектуры базы данных

- Создан новый Docker контейнер PostgreSQL на порту 5489
- Создан новый Docker контейнер Redis на порту 6396
- Реализована архитектура на основе TypeORM с паттернами Repository и Service

### 2. Созданные сущности (Entities)

- **BaseEntity** - базовая сущность с UUID
- **AuditableEntity** - расширенная сущность с аудитом
- **User** - пользователи системы
- **Form** - формы заявок
- **FormField** - поля форм
- **Submission** - заявки
- **SubmissionHistory** - история изменений заявок
- **AdminToken** - токены администраторов
- **SubmissionFieldValue** - значения полей заявок

### 3. Сервисный слой

Созданы сервисы для работы с данными:
- **UserService** - управление пользователями
- **FormService** - управление формами
- **SubmissionService** - управление заявками
- **CacheService** - кеширование через Redis

### 4. Репозитории

Реализованы репозитории с поддержкой:
- Пагинации
- Фильтрации
- Сортировки
- Кеширования
- Денормализованных полей для оптимизации

### 5. Миграция данных

- Создан скрипт импорта из JSON бэкапа
- Успешно импортированы:
  - 70 пользователей
  - 1 форма
  - 16 полей формы
  - 15 заявок

### 6. Конфигурация

#### PostgreSQL
- Host: localhost
- Port: 5489
- Database: beton_crm
- User: beton_user
- Password: beton_password

#### Redis
- Host: localhost
- Port: 6396

#### Переменные окружения (.env)
```
DB_HOST=localhost
DB_PORT=5489
DB_USERNAME=beton_user
DB_PASSWORD=beton_password
DB_NAME=beton_crm
REDIS_HOST=localhost
REDIS_PORT=6396
```

### 7. Запуск проекта

1. Запустить контейнеры:
```bash
docker-compose up -d postgres redis
```

2. Запустить сервер:
```bash
npm run dev
```
