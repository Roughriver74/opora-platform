# Скрипт синхронизации БД с DEV сервером

## Описание

Отдельный скрипт для синхронизации локальной базы данных PostgreSQL с dev сервером.

## Использование

```bash
./scripts/sync-db-to-dev.sh
```

## Что делает скрипт

1. **Проверяет локальную БД** - ищет Docker PostgreSQL или системную PostgreSQL
2. **Запрашивает пароль** - если пароль не найден в .env файле
3. **Создает дамп** локальной БД с полной статистикой
4. **Показывает статистику** локальной БД (пользователи, заявки, формы)
5. **Запрашивает подтверждение** перед перезаписью БД на сервере
6. **Создает резервную копию** текущей БД на сервере
7. **Восстанавливает БД** на dev сервере
8. **Показывает статистику** восстановленной БД

## Безопасность

- ✅ Создает резервную копию БД на сервере перед восстановлением
- ✅ Запрашивает подтверждение перед перезаписью
- ✅ Показывает статистику до и после синхронизации
- ✅ Использует безопасное подключение по SSH

## Требования

- Локальная БД PostgreSQL (Docker или системная)
- Доступ к dev серверу по SSH
- Пароль для подключения к БД

## Переменные окружения

Скрипт автоматически загружает переменные из `.env` файла:

- `POSTGRES_PASSWORD` - пароль для подключения к БД

Если пароль не найден в .env, скрипт запросит его интерактивно.
