# 🗄️ Система управления базой данных в реальном времени

## 📋 Обзор

Новая система для удобного редактирования базы данных через интерфейс админ-панели с возможностью редактирования в реальном времени.

## 🎯 Возможности

### ✨ Основные функции

- **Inline редактирование** - Дважды кликните на любую ячейку для редактирования
- **Автосохранение** - Изменения сохраняются при нажатии Enter или потере фокуса
- **Поиск и фильтрация** - Глобальный поиск по всем полям таблицы
- **Сортировка** - Кликните на заголовок колонки для сортировки
- **Индикаторы состояния** - Визуальная обратная связь при сохранении
- **Уведомления** - Мгновенные уведомления об успехе или ошибках

### 📊 Поддерживаемые таблицы

#### 1. **Поля форм**

Редактирование полей формы:

- `name` - Название поля
- `label` - Отображаемое название
- `type` - Тип поля
- `order` - Порядок отображения ⭐ (основное назначение)
- `required` - Обязательность поля
- `bitrixFieldId` - Идентификатор в Битрикс24

#### 2. **Пользователи**

Управление пользователями:

- `firstName` - Имя
- `lastName` - Фамилия
- `email` - Email адрес
- `role` - Роль (admin/user)
- `bitrix_id` - ID в Битрикс24

## 🚀 Как использовать

### Доступ к системе

1. Войдите в админ-панель как администратор
2. Перейдите на вкладку **"База данных"** (3-я вкладка)
3. Выберите нужную таблицу в подвкладках

### Редактирование данных

1. **Начать редактирование**: Дважды кликните на нужную ячейку
2. **Сохранить изменения**: Нажмите `Enter` или кликните вне поля
3. **Отменить редактирование**: Нажмите `Escape`

### Поиск и фильтрация

- Используйте поле поиска для фильтрации по любому полю
- Поиск работает в реальном времени
- Счетчик показывает количество найденных записей

### Сортировка

- Кликните на заголовок любой колонки для сортировки
- Повторный клик меняет направление сортировки
- Стрелки показывают текущее направление сортировки


### Технологии

- **TanStack Table** - Современная библиотека для таблиц
- **Material-UI** - Компоненты интерфейса
- **React Hooks** - Управление состоянием
- **TypeScript** - Типизация

### API эндпоинты

- `GET /api/form-fields` - Получение полей форм
- `PUT /api/form-fields/:id` - Обновление поля формы
- `GET /api/users` - Получение пользователей
- `PUT /api/users/:id` - Обновление пользователя

## 🎛️ Конфигурация

### Добавление новой таблицы

1. **Создайте хук** в `hooks/` для работы с данными:

```typescript
export const useMyTable = () => {
	// логика загрузки и обновления данных
}
```

2. **Добавьте конфигурацию колонок** в `index.tsx`:

```typescript
const myTableColumns = [
	{
		accessorKey: 'fieldName',
		header: 'Заголовок',
		size: 150,
	},
	// ... другие колонки
]
```

3. **Добавьте Tab и TabPanel** в основной компонент
4. **Подключите хук и передайте данные** в DatabaseTable

### Настройка колонок

- `accessorKey` - имя поля в данных
- `header` - заголовок колонки
- `size` - ширина колонки в пикселях
- `cell` - функция для кастомного отображения значения

## 🛡️ Безопасность

## 🔧 Устранение неполадок

### Проблемы с загрузкой данных

1. Проверьте подключение к серверу
2. Убедитесь что API эндпоинты доступны
3. Проверьте права доступа пользователя

### Проблемы с сохранением

1. Проверьте формат данных
2. Убедитесь в корректности валидации
3. Проверьте логи сервера

### Производительность

- Таблица поддерживает большие объемы данных
- Виртуализация строк для оптимизации
- Ленивая загрузка при необходимости

## 📝 Примеры использования

### Изменение порядка полей формы

1. Перейдите в таблицу "Поля форм"
2. Найдите нужное поле через поиск
3. Дважды кликните на колонку "Порядок"
4. Введите новое значение (например: 150)
5. Нажмите Enter для сохранения

### Изменение роли пользователя

1. Перейдите в таблицу "Пользователи"
2. Найдите пользователя по email
3. Дважды кликните на колонку "Роль"
4. Измените на "admin" или "user"
5. Изменения сохранятся автоматически
