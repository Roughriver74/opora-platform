- ✅ Задержки: 1s, 2s, 4s между попытками
- ✅ Специфичный timeout 15 секунд для запросов к Битрикс24
- ✅ Информативные сообщения с номером попытки
- ✅ Кнопка "Повторить" для ручного запуска
- ✅ Обработка различных типов ошибок (timeout, 502, 503)
- ✅ Исправлены все TypeScript lint ошибки

**Код:**
```typescript
const loadCategories = async (retryCount = 0) => {
  const maxRetries = 2
  
  try {
    const response = await api.get('/api/forms/bitrix/deal-categories', {
      timeout: 15000, // Специфичный timeout
    })
    // ... обработка
  } catch (err: any) {
    if (err.code === 'ECONNABORTED' || err.message?.includes('timeout')) {
      const shouldRetry = retryCount < maxRetries
      if (shouldRetry) {
        const delay = Math.min(1000 * Math.pow(2, retryCount), 5000)
        setTimeout(() => loadCategories(retryCount + 1), delay)
      }
    }
  }
}
```

**Результат:**
- ⏱️ Автоматически повторяет запрос при ошибке
- 📊 Показывает текущую попытку: "(попытка 1/3)"
- 🔄 Можно запустить вручную кнопкой "Повторить"
- ⚠️ Разные сообщения для разных ошибок
