# Административные функции

## Редактирование названий разделов

### Описание

Администраторы теперь могут редактировать названия разделов формы прямо в интерфейсе пользователя.

### Как использовать

1. **Войдите как администратор**

   - Используйте административный пароль для входа в систему

2. **Откройте форму**

   - Перейдите на главную страницу с формой заказа бетона

3. **Найдите раздел для редактирования**

   - В режиме администратора рядом с названием каждого раздела появится иконка редактирования (карандаш)

4. **Редактируйте название**

   - Нажмите на иконку редактирования рядом с названием раздела
   - Введите новое название в появившемся поле ввода
   - Нажмите кнопку "Сохранить" (галочка) или клавишу Enter для сохранения
   - Нажмите кнопку "Отменить" (крестик) или клавишу Escape для отмены

5. **Подтверждение изменений**
   - После успешного сохранения появится уведомление
   - Страница автоматически перезагрузится для отображения изменений

### Технические детали

- Функция доступна только администраторам
- Изменения сохраняются в базе данных
- Редактируются поля типа "header" (заголовки разделов)
- Изменения применяются ко всем формам, использующим данный раздел

### Ограничения

- Нельзя оставить пустое название раздела
- Изменения применяются глобально для всех пользователей
- Требуется перезагрузка страницы для отображения изменений

### Безопасность

- Функция защищена проверкой роли пользователя
- Все запросы на изменение проходят через авторизацию
- Логируются все изменения для аудита
