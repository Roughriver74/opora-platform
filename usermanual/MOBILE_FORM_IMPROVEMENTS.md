# Доработка мобильной формы

## Обзор изменений

Обновлена мобильная форма `SimpleMobileBetoneForm.tsx` с фокусом на более компактный дизайн для телефонов и добавлены индикаторы заполненности секций.

## ✨ Новые возможности

### 1. Более компактный дизайн для телефонов

#### Размеры элементов:

- **Touch targets**: уменьшены с 48px до 44px
- **Анимации**: ускорены с 200ms до 150ms
- **Padding**: уменьшен с 2 до 1.5
- **Spacing**: уменьшен с 1 до 0.5

#### Шрифты:

- **Основной текст**: уменьшен с 0.9rem до 0.875rem
- **Заголовки**: изменены с `h6` на `subtitle2`
- **Кнопки**: добавлен размер `small`
- **Высота кнопок**: уменьшена до 36px на мобильных

#### Отступы:

- **Общие отступы**: уменьшены по всей форме
- **Секции**: более компактные отступы между элементами
- **Заголовки**: уменьшены отступы заголовков

### 2. Индикаторы заполненности секций

#### Функция подсчета заполненности:

```typescript
const calculateSectionCompleteness = (
	section: any,
	values: Record<string, any>
) => {
	// Возвращает:
	// - filled: количество заполненных полей
	// - total: общее количество полей
	// - required: количество обязательных полей
	// - requiredFilled: количество заполненных обязательных полей
	// - percentage: процент заполненности
	// - isComplete: все обязательные поля заполнены
}
```

#### Компактная версия для мобильных:

- **Иконка статуса**: ✅ (заполнено) или ⭕ (не заполнено)
- **Счетчик**: "3/5" (заполнено/всего)
- **Цвет**: зеленый для завершенных, серый для незавершенных

#### Полная версия для планшетов:

- **Чип с процентами**: "75%" с цветовой индикацией
- **Детальный счетчик**: "3/5" полей
- **Цвета**: зеленый (завершено), оранжевый (>50%), серый (<50%)

#### Прогресс-бар:

- **Расположение**: под заголовком каждой секции
- **Высота**: 2px для ненавязчивости
- **Цвет**: динамический (зеленый для завершенных, синий для незавершенных)

### 3. Визуальные улучшения

#### Тени и границы:

- **Мобильные**: убраны тени (`elevation: 0`), добавлены границы
- **Планшеты**: сохранены легкие тени (`elevation: 1`)

#### Иконки:

- **Размер**: уменьшен до 20px для компактности
- **Отступы**: оптимизированы для мобильных

#### Заголовки:

- **Высота**: уменьшена до 40px на мобильных
- **Шрифт**: более компактный `subtitle2`
- **Отступы**: оптимизированы для экономии места

## 🚀 Использование

Форма автоматически определяет тип устройства и адаптируется:

```typescript
// Автоматическое определение
const isMobile = useMediaQuery(theme.breakpoints.down('sm'))

// Условная отрисовка
{
	isMobile ? (
		<SectionCompletionIndicator compact={true} />
	) : (
		<SectionCompletionIndicator compact={false} />
	)
}
```
