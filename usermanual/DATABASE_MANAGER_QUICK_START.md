# 🚀 Быстрый старт: Система управления базой данных

## 📍 Как попасть в систему

1. **Войдите в админ-панель** как администратор
2. **Перейдите на вкладку "База данных"** (3-я вкладка между "Формы" и "Битрикс24")

## ⚡ Основные действия

### 🎯 Редактирование порядка полей (главная задача)

**Вкладка "Поля форм":**

1. Найдите нужное поле через поиск 🔍
2. Смотрите в колонку **"Раздел"** - к какому разделу принадлежит поле
3. Дважды кликните на колонку **"Порядок"**
4. Введите новое значение согласно правилам нумерации
5. Нажмите `Enter` или кликните в другое место
6. ✅ Поле автоматически сохранится

### 🔧 Автоматическая нормализация порядка

**Если нужно исправить все поля сразу:**

1. Нажмите кнопку **"Нормализовать порядок"**
2. Просмотрите планируемые изменения в диалоге
3. Нажмите **"Применить изменения"**
4. ✅ Система автоматически исправит всю нумерацию

### 📋 Правила нумерации полей

- **Разделы (заголовки)**: 100, 200, 300, 400...
- **Поля в разделе 100**: 101, 102, 103, 104...
- **Поля в разделе 200**: 201, 202, 203, 204...
- **Поля без раздела**: 1, 2, 3, 4...

### 👤 Управление пользователями

**Вкладка "Пользователи":**

- Измените **роль** (admin/user)
- Обновите **имя**, **фамилию**
- Измените **email**
- Обновите **Битрикс ID**

## 💡 Удобные функции

### 🔍 Поиск и фильтрация

- Введите любой текст в поле поиска
- Поиск работает по всем колонкам
- Счетчик показывает количество найденных записей

### 📊 Сортировка

- Кликните на любой заголовок колонки
- Стрелки показывают направление сортировки
- Повторный клик меняет направление

### 🔔 Уведомления

- ✅ Зеленое уведомление при успешном сохранении
- ❌ Красное уведомление при ошибке
- Автоматически исчезают через несколько секунд

## 🎮 Горячие клавиши

- **Double Click** - начать редактирование
- **Enter** - сохранить изменения
- **Escape** - отменить редактирование

## 🔧 Решение проблем

### Поле не сохраняется?

1. Проверьте формат данных (числа для порядка)
2. Убедитесь что поле не пустое
3. Проверьте подключение к серверу

### Не видно изменений?

1. Обновите страницу (`F5`)
2. Проверьте что сервер запущен
3. Посмотрите в консоль браузера (`F12`)

## 📋 Частые задачи

### Изменить порядок нескольких полей:

1. Отсортируйте по колонке "Порядок"
2. По очереди редактируйте каждое поле
3. Используйте логичную нумерацию (100, 200, 300...)

### Найти поле по названию:

1. Введите часть названия в поиск
2. Система найдет все совпадения
3. Редактируйте найденное поле

---

**💡 Совет:** Всегда используйте логичную нумерацию порядка полей с запасом (100, 200, 300), чтобы потом можно было вставить поля между ними (150, 250).
