# Интеграция Elasticsearch для получения наименований

## Описание

Добавлена функциональность для получения человекочитаемых названий компаний и продуктов из индексов Elasticsearch вместо отображения только ID.

## Реализованные компоненты

### 1. Сервис ElasticsearchService (`client/src/services/elasticsearchService.ts`)

Новый сервис для работы с Elasticsearch API:

- `getCompanyNameById(id)` - получение названия компании по ID
- `getProductNameById(id)` - получение названия продукта по ID
- `getCompanyNamesByIds(ids[])` - массовое получение названий компаний
- `getProductNamesByIds(ids[])` - массовое получение названий продуктов
- Встроенное кэширование с TTL 5 минут
- Автоматическая очистка кэша при превышении лимита

### 2. Обновленный компонент MySubmissions

#### Новые состояния:

- `companyNames` - кэш названий компаний
- `productNames` - кэш названий продуктов

#### Новые функции:

- `loadNamesForSubmissions()` - загрузка названий для всех заявок
- `getCompanyName()` - получение названия компании по полю формы
- `getProductName()` - получение названия продукта по полю формы

#### Обновленное отображение:

- Компании теперь показывают названия вместо ID
- Продукты (бетон) показывают названия вместо ID
- Fallback на ID если название не найдено

## Используемые поля формы

- `field_1750266840204` - поле компании
- `field_1750264442280` - поле бетона/продукта

## API Endpoints

Сервис использует существующие Elasticsearch endpoints:

- `POST /api/search/companies` - поиск компаний
- `POST /api/search/products` - поиск продуктов

## Оптимизация

1. **Кэширование**: Названия кэшируются на 5 минут
2. **Массовые запросы**: Загружаются все уникальные ID за один раз
3. **Параллельная загрузка**: Компании и продукты загружаются параллельно
4. **Fallback**: При ошибке показывается ID вместо названия

## Пример использования

```typescript
// Получение названия компании
const companyName = await elasticsearchService.getCompanyNameById('12345')

// Массовое получение названий
const names = await elasticsearchService.getCompanyNamesByIds([
	'123',
	'456',
	'789',
])
```

## Совместимость

- Работает с существующими данными
- Не нарушает текущую функциональность
- Graceful degradation при недоступности Elasticsearch
