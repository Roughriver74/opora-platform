### Автоматический запуск (рекомендуется)

```bash
./start.sh
```

Скрипт автоматически:

- ✅ Проверит наличие Node.js и npm
- ✅ Установит зависимости
- ✅ Скомпилирует сервер
- ✅ Запустит сервер на порту 5001
- ✅ Запустит клиент на порту 3000

### Ручной запуск

#### 1. Запуск сервера

```bash
cd server
npm install
npm run build
npm run dev
```

#### 2. Запуск клиента (в новом терминале)

```bash
cd client
npm install
npm start
```

## 🎮 Как пользоваться

### Для администраторов:

1. Войдите под админ-аккаунтом
2. Перейдите в раздел "Администрирование"
3. Используйте вкладки для управления системой:
   - **Формы** - создавайте формы заказов
   - **Заявки** - просматривайте и обрабатывайте заявки
   - **Пользователи** - управляйте пользователями
   - **Битрикс24** - настраивайте интеграцию

### Для пользователей:

1. Войдите под пользовательским аккаунтом
2. Заполните форму заказа на главной странице
3. Отслеживайте статус в разделе "Мои заявки"

## 🐛 Решение проблем

### Если не запускается:

1. Проверьте версию Node.js (требуется 16+)
2. Убедитесь, что MongoDB запущен
3. Проверьте файл server/.env
4. Очистите кэш: `npm cache clean --force`

### Если ошибки компиляции:

```bash
cd server && npm run build
cd ../client && npm run build
```

### Если проблемы с авторизацией:

1. Очистите localStorage в браузере
2. Проверьте, создан ли админ-пользователь
3. Проверьте JWT_SECRET в .env
